# Discord Server Cloner (Bot & Selfbot)

Bu araç, bir Discord sunucusundaki rolleri, kategorileri ve kanalları hızlıca başka bir sunucuya kopyalamanızı sağlar. Hem **Standart Discord Botları** hem de **Kullanıcı Hesapları (Selfbot)** ile çalışacak şekilde tasarlanmıştır.

## Özellikler
- 💻 Şık ve interaktif CMD arayüzü.
- 🤖 Hem Bot Token hem de User Token (Selfbot) desteği.
- 🗂️ Sunucu rollerini (renkleri ve izinleriyle) kopyalar.
- 📁 Kategorileri, metin ve ses kanallarını kopyalar.
- 🗑️ Kopyalamadan önce hedef sunucuyu otomatik temizler.
